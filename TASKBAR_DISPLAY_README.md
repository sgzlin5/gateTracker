# 任务栏嵌入式显示器功能说明

## 功能概述

本项目新增了Windows任务栏嵌入式价格显示功能，可以将实时价格信息直接显示在任务栏附近，让价格信息更加醒目和易于查看。

## 主要特性

### 1. 自动定位
- 自动检测Windows任务栏的位置（底部、顶部、左侧、右侧）
- 自动将价格显示器定位到任务栏附近
- 每3秒自动检测任务栏位置变化并调整窗口位置

### 2. 紧凑显示
- 高度与任务栏相近（36px）
- 实时显示价格和涨跌方向
- 使用颜色标识涨跌（绿色上涨、红色下跌、白色平盘）
- 格式：`交易对名称:价格↑ | 交易对名称:价格↓`

### 3. 可拖动窗口
- 支持鼠标左键拖动窗口到任意位置
- 拖动后窗口位置将被锁定，不再自动重新定位
- 重新显示窗口时会恢复到默认位置

### 4. 智能集成
- 与主程序无缝集成
- 自动接收价格更新数据
- 可通过右键菜单控制显示/隐藏
- 启用时主窗口自动最小化，关闭时主窗口自动恢复

## 使用方法

### 方法一：在主程序中使用

1. **启动程序**
   ```bash
   python main.py
   ```

2. **显示任务栏显示器**
   - 在主窗口上点击右键
   - 选择"显示任务栏显示器"选项
   - 任务栏显示器将自动显示在任务栏附近
   - 主窗口会自动最小化到任务栏

3. **隐藏任务栏显示器**
   - 再次右键点击主窗口（如果已最小化，请在任务栏中恢复窗口）
   - 取消选择"显示任务栏显示器"选项
   - 主窗口会自动恢复正常显示

4. **拖动任务栏显示器**
   - 使用鼠标左键按住任务栏显示器
   - 拖动到任意位置后释放
   - 拖动后窗口位置将被锁定，不再自动重新定位

5. **直接关闭任务栏显示器**
   - 点击任务栏显示器窗口的关闭按钮（如果有）
   - 主窗口会自动恢复正常显示

### 方法二：使用测试脚本

1. **运行测试脚本**
   ```bash
   python test_taskbar.py
   ```

2. **测试功能**
   - 点击"显示任务栏显示器"按钮查看效果
   - 点击"更新价格数据"按钮测试价格更新
   - 点击"切换任务栏显示器"按钮切换显示状态

## 效果展示

### 任务栏在底部（默认）
显示器会显示在任务栏上方右侧

### 任务栏在顶部
显示器会显示在任务栏下方右侧

### 任务栏在左侧
显示器会显示在任务栏右侧底部

### 任务栏在右侧
显示器会显示在任务栏左侧底部

## 显示效果示例

```
XAUUSD:2345.67↑  |  NAS100:18900.32↓
```

- **绿色**：价格上涨（↑）
- **红色**：价格下跌（↓）
- **白色**：价格持平（→）

## 技术实现

### 核心组件

1. **TaskbarPositionHelper**
   - 使用Win32 API获取任务栏位置
   - 检测任务栏在屏幕的哪个边

2. **TaskbarPriceDisplay**
   - 创建无边框、置顶的紧凑窗口
   - 定时器定期检查任务栏位置
   - 实时更新价格显示
   - 支持鼠标拖动窗口

3. **TaskbarDisplayController**
   - 管理显示器的生命周期
   - 提供显示/隐藏/切换接口
   - 处理价格数据更新

### 依赖项

- `pywin32>=306` - Windows API访问
- `PyQt6>=6.5.0` - GUI框架

## 注意事项

1. **任务栏位置变化**
   - 移动任务栏时，显示器会在3秒内自动调整位置
   - 建议保持任务栏位置稳定以获得最佳显示效果

2. **窗口拖动**
   - 使用鼠标左键拖动窗口到任意位置
   - 拖动后窗口位置锁定，不再自动重新定位
   - 重新显示窗口时会恢复到默认位置

3. **多显示器支持**
   - 当前版本主要支持主显示器
   - 在多显示器环境中，显示器会根据主显示器的任务栏定位

4. **系统兼容性**
   - 仅支持Windows系统
   - 需要Windows 10及以上版本

5. **性能考虑**
   - 每个显示器每3秒检查一次位置（CPU占用极低）
   - 价格更新不会影响显示器的定位

## 故障排除

### 问题1：任务栏显示器不显示

**解决方案：**
- 确保已安装pywin32：`pip install pywin32`
- 检查是否有其他程序阻挡了显示区域
- 尝试重新启动程序

### 问题2：位置不正确

**解决方案：**
- 检查任务栏是否被锁定
- 尝试移动任务栏栏后再放回原位
- 等待3秒让自动定位生效
- 可以手动拖动窗口到合适位置

### 问题3：无法关闭任务栏显示器

**解决方案：**
- 在主窗口右键菜单中关闭显示器
- 检查配置文件是否正常
- 尝试重启程序

### 问题4：拖动窗口后位置不正确

**解决方案：**
- 这是正常行为，拖动后窗口位置会被锁定
- 如果需要恢复自动定位，关闭并重新打开显示器
