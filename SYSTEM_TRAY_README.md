# 系统托盘功能说明

## 功能概述

本程序现在支持将图标隐藏到右下角的系统托盘区域，提供更加便捷的操作方式。程序启动后会自动创建系统托盘图标，您可以随时通过托盘图标控制程序。

## 主要特性

### 1. 系统托盘图标
- 程序启动后自动在系统托盘区域显示图标
- 鼠标悬停显示当前价格信息
- 支持左键单击和双击操作
- 支持右键菜单访问所有功能

### 2. 便捷操作
- **左键单击**：显示/隐藏主窗口
- **双击**：显示主窗口并激活
- **右键菜单**：访问所有程序功能

### 3. 关闭行为
- 点击主窗口的关闭按钮时，窗口会隐藏到托盘而不是退出程序
- 要退出程序，请通过托盘图标的右键菜单选择"退出"

## 托盘右键菜单功能

### 显示/隐藏主窗口
- 点击可切换主窗口的显示状态
- 如果主窗口可见，会隐藏窗口
- 如果主窗口隐藏，会显示并激活窗口

### 显示任务栏显示器
- 可勾选/取消勾选以启用或禁用任务栏显示器
- 启用后主窗口会自动最小化
- 禁用后主窗口会自动恢复正常显示

### 切换主题
- 在暗色和亮色主题之间切换
- 切换后立即应用到所有窗口

### 设置窗口大小...
- 打开设置对话框
- 可以调整窗口宽度和高度
- 可以设置窗口是否始终置顶

### 退出
- 完全退出程序
- 会保存所有配置和窗口状态
- 关闭所有连接和资源

## 使用方法

### 启动程序
```bash
python main.py
```

### 基本操作流程

1. **启动程序**
   - 程序启动后会自动显示系统托盘图标
   - 主窗口也会同时显示

2. **隐藏主窗口到托盘**
   - 方法一：点击主窗口的关闭按钮（X）
   - 方法二：左键单击托盘图标
   - 方法三：在托盘右键菜单选择"隐藏主窗口"

3. **从托盘恢复主窗口**
   - 方法一：左键单击托盘图标
   - 方法二：双击托盘图标
   - 方法三：在托盘右键菜单选择"显示主窗口"

4. **使用任务栏显示器**
   - 在托盘右键菜单勾选"显示任务栏显示器"
   - 主窗口会自动最小化
   - 价格信息会显示在任务栏附近

5. **退出程序**
   - 在托盘右键菜单选择"退出"
   - 或直接关闭所有窗口（如果托盘不可用）

## 系统托盘图标说明

### 默认图标
- 如果提供了自定义图标文件（`icon.ico`、`icon.png`等），会使用自定义图标
- 如果没有提供，会使用系统默认的应用程序图标

### 工具提示
- 鼠标悬停在托盘图标上时显示
- 内容格式：`Gate Price Tracker - XAUUSD 2345.67 ↑ | NAS100 18900.32 ↓`
- 实时更新显示最新价格信息

## 技术实现

### 核心组件

1. **SystemTrayController**
   - 管理系统托盘图标的创建和生命周期
   - 提供信号机制与主窗口通信
   - 处理托盘图标的激活事件

2. **信号系统**
   - `show_window_requested`: 请求显示主窗口
   - `hide_window_requested`: 请求隐藏主窗口
   - `toggle_taskbar_display_requested`: 切换任务栏显示器
   - `toggle_theme_requested`: 切换主题
   - `show_settings_requested`: 显示设置对话框
   - `exit_requested`: 退出程序

3. **PriceWindow集成**
   - 初始化时创建托盘控制器
   - 连接所有信号到相应的处理方法
   - 更新托盘状态以反映当前程序状态

### 依赖项

- `PyQt6>=6.5.0` - QSystemTrayIcon支持
- 无需额外依赖（使用PyQt6内置功能）

## 注意事项

### 1. 系统兼容性
- 仅支持Windows系统
- 需要Windows 10及以上版本
- 某些精简版Windows可能不支持系统托盘

### 2. 托盘可用性检查
- 程序启动时会检查系统托盘是否可用
- 如果系统托盘不可用，程序仍可正常运行
- 主窗口的关闭行为会正常退出程序

### 3. 自动隐藏行为
- 关闭主窗口时会自动隐藏到托盘（如果托盘可用）
- 这是设计行为，不是bug
- 要退出程序请使用托盘菜单的"退出"选项

### 4. 性能考虑
- 托盘图标几乎不占用系统资源
- 工具提示更新与价格更新同步
- 不会影响程序的实时性能

### 5. 多显示器支持
- 托盘图标始终显示在系统托盘区域
- 主窗口可以在任意显示器上显示
- 窗口状态会正确保存和恢复

## 故障排除

### 问题1：托盘图标不显示

**可能原因：**
- 系统托盘功能被禁用
- 使用了不支持托盘的系统版本

**解决方案：**
- 检查系统托盘设置是否启用
- 确认使用的是Windows 10或更高版本
- 查看控制台输出的错误信息

### 问题2：关闭窗口后程序不退出

**这是正常行为：**
- 程序设计为关闭窗口时隐藏到托盘
- 要完全退出程序，请使用托盘菜单的"退出"选项

### 问题3：托盘菜单功能不响应

**可能原因：**
- 信号连接失败
- 主窗口被意外销毁

**解决方案：**
- 重启程序
- 查看控制台输出的错误信息
- 检查是否有其他程序冲突

### 问题4：工具提示不更新

**可能原因：**
- WebSocket连接断开
- 价格更新功能异常

**解决方案：**
- 检查网络连接
- 查看WebSocket连接状态
- 重启程序

## 自定义图标

### 支持的图标格式
- ICO格式（推荐）：`icon.ico`
- PNG格式：`icon.png`

### 图标放置位置
程序会按以下顺序查找图标文件：
1. `icon.ico` / `icon.png`（当前目录）
2. `assets/icon.ico` / `assets/icon.png`
3. `resources/icon.ico` / `resources/icon.png`

### 图标设计建议
- 使用16x16、32x32、48x48等多尺寸
- 支持透明背景
- 在不同背景下保持清晰可见
